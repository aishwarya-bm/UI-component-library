Introduction

UI Cookbook is a component library which comes with a broad spectrum of utility classes. The library aims at building UI quickly by focussing on the functionality by providing the predefined classes. These classes help in building user interfaces for your websites without reinventing the wheel.


To introduce the UI Cookbook library to your project, copy the link given below and paste it in the <head> tag of your html, above all other style sheets. 

                        
                        https://ui-cookbook.netlify.app/stylesheets/components/main.css
